# Homebrew Tap for yepzdk Tools

This is a Homebrew tap for tools developed by yepzdk.

## Installation

```bash
brew tap yepzdk/tools
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| `csm` | Monitor Claude Code sessions |

## Usage

Install a formula:

```bash
brew install csm
```

## How It Works

Formulae are automatically added and updated by GoReleaser when new releases are published.
